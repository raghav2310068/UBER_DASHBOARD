# 🚖 Uber Trip Analytics Dashboard | Power BI

An interactive **Power BI dashboard** designed to analyze Uber trip data, providing insights into ride performance, revenue, customer behavior, vehicle performance, and operational efficiency.

The dashboard enables users to explore key business metrics through interactive visualizations, filters, and drill-down analysis for data-driven decision making.

# 📊 Dashboard Features

## 🏠 Landing Page
- Interactive navigation menu
- Professional dashboard introduction
- Quick access to all report pages

---

## 📈 Overview Dashboard

Provides an overall summary of Uber operations.

### KPI Cards
- Total Completed Rides
- Total Rides Lost
- Total Revenue
- Total Distance Covered
- Average Trip Distance

### Visualizations
- Monthly/Quarterly Ride Trends
- Revenue Trend Analysis
- Revenue by Vehicle Type
- Rides Lost by Vehicle
- Ride Status Distribution
  - Completed
  - Cancelled
  - Incomplete

---

## 🚗 Vehicle Dashboard

Analyze the performance of every vehicle category.

### Metrics Included

- Vehicle Type
- Total Customers
- Revenue Generated
- Completed Rides
- Completion Percentage
- Monthly Booking Trend

Vehicle Categories:

- Auto
- Bike
- Go Mini
- Go Sedan
- Premier Sedan
- Uber XL

---

## 💰 Revenue Dashboard

Revenue-focused analytics including:

### Revenue Trend
- Monthly Revenue Analysis

### Revenue by Vehicle

Compare revenue generated by each vehicle category.

### Revenue by Payment Method

Supported payment methods:

- UPI
- Cash
- Uber Wallet
- Credit Card
- Debit Card

### Top Revenue Generating Customers

Identify customers contributing the highest revenue.

# 🚀 How to Use

1. Clone the repository.

```bash
git clone https://github.com/yourusername/Uber-Trip-Analytics.git
```

2. Open

```
Uber.pbix
```

using **Power BI Desktop**.

3. Explore different pages using the navigation buttons.

4. Use filters and slicers to interact with the dashboard.
